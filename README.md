# 数独比赛计时器

一个专为数独比赛设计的计时器应用，支持多轮次管理、语音提醒和自定义样式。

## 功能特性

- **多轮次管理**：支持添加多个比赛轮次和场间休息
- **语音提醒**：支持中英文语音包，自动提醒剩余时间
  - 常规模式：剩余 5 分钟、1 分钟、时间到
  - 轮转模式：按设定间隔提醒"转"
- **自定义样式**：可自定义背景颜色、文字颜色、字体和字号
- **快捷键支持**：空格暂停/继续，ESC 退出，F 全屏，S 跳过
- **Electron 打包**：可打包为 Windows 便携版 exe 文件

## 项目结构

```
timer/
├── index.html          # 主页面 HTML
├── script.js           # 核心逻辑代码
├── styles.css          # 样式文件
├── main.js             # Electron 主进程
├── package.json        # 项目配置
├── audios/             # 语音文件
│   ├── zh/            # 中文语音
│   └── en/            # 英文语音
├── dist/              # 打包输出目录
└── 启动计时器.bat      # Windows 启动脚本
```

## 快速开始

### 方式一：直接在浏览器中运行

双击 `index.html` 文件，使用浏览器打开即可使用。

### 方式二：使用 Electron 运行

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

### 方式三：打包为 Windows 可执行文件

```bash
npm run build
```

打包完成后，可在 `dist/` 目录下找到 `数独比赛计时器.exe`。

## 使用说明

### 1. 基础设置

- **赛会名称**：设置全局赛会名称（所有轮次统一使用）
- **语音包选择**：选择中文或英文语音包
- **加载语音包**：点击按钮加载音频文件

### 2. 轮次管理

- **添加比赛轮次**：设置轮次名称、时长（分钟）、提醒类型
- **添加场间休息**：设置休息时长
- **提醒类型**：
  - 常规轮次：剩余 5 分钟和 1 分钟时提醒
  - 轮转模式：按设定间隔提醒

### 3. 样式设置

- 背景颜色和文字颜色
- 字体选择
- 赛会名称、轮次名称、倒计时数字的字号

### 4. 计时器控制

| 按钮 | 功能 |
|------|------|
| 暂停 | 暂停当前计时 |
| 继续 | 恢复计时 |
| 跳过当前 | 跳到下一轮次 |
| 全部停止 | 停止所有计时 |
| 全屏 | 进入/退出全屏 |
| 退出 | 返回设置界面 |

### 5. 快捷键

| 快捷键 | 功能 |
|--------|------|
| 空格 | 暂停/继续 |
| ESC | 退出计时界面 |
| F | 切换全屏 |
| S | 跳过当前轮次 |

## 开发相关

### 技术栈

- **前端**：HTML5、CSS3、JavaScript (ES6+)
- **音频**：Web Audio API + HTML5 Audio
- **桌面端**：Electron 28

### 音频文件格式

语音文件存放在 `audios/` 目录：

- `audios/zh/` - 中文语音
  - `剩余五分钟.mp3`
  - `剩余一分钟.mp3`
  - `时间到.mp3`
  - `转.mp3`
- `audios/en/` - 英文语音
  - `5_minutes_left.mp3`
  - `1_minute_left.mp3`
  - `Time's_up.mp3`
  - `Turn.mp3`

如需替换音频，只需替换对应文件即可。

### 配置文件

设置会自动保存到浏览器 localStorage，刷新页面后会自动加载上次保存的配置。
