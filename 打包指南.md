# 数独比赛计时器 - 打包指南

## 准备工作

### 1. 安装 Node.js
如果还没有安装 Node.js，请先下载安装：
- 下载地址：https://nodejs.org/zh-cn/
- 选择 **LTS 长期支持版**（推荐）
- 安装时一路"下一步"即可

### 2. 验证安装
打开命令行（Win+R，输入 cmd），输入：
```
node -v
```
如果显示版本号（如 v18.x.x），说明安装成功。

---

## 打包步骤

### 步骤 1：清理之前的安装
如果之前安装失败，先删除文件夹：
```
cd 你的文件夹路径\数独比赛计时器
rmdir /s /node_modules
```

### 步骤 2：安装依赖
运行：
```
npm install
```

### 步骤 3：打包成 EXE
运行：
```
npm run build
```

### 步骤 4：找到 EXE 文件
打包完成后，在 `dist` 文件夹中找到 `数独比赛计时器.exe`，这就是打包好的程序！

---

## 常见问题

### Q: npm install 报错 ETIMEDOUT
**A:** 这是网络超时问题。尝试：
1. 检查网络是否正常
2. 多试几次（有时候网络不稳定）
3. 或者尝试使用手机热点

### Q: 安装太慢
**A:** Electron 需要下载较大的文件，可能需要 5-10 分钟，请耐心等待。

### Q: 打包失败
**A:** 
1. 删除 `node_modules` 文件夹
2. 删除 `package-lock.json`
3. 重新运行 `npm install`
